### 📌 Overview

The **Institute of Drone Education and Aerospace (IDEAAS)** website is a modern, responsive platform built using **Next.js** and **Tailwind CSS**.

It is designed to showcase drone training programs, courses, and services through an engaging and user-friendly interface. The project focuses on **performance optimization, mobile-first design, and seamless navigation**, ensuring a smooth experience across all devices.

The platform includes dynamic course sections, an interactive gallery, and functional contact forms to enhance user engagement and accessibility.
